Joala Build-Tools

  • Last Published: 2017-07-31|
  • Version: 1.2.0|
  • Joala BOM/
  • Joala Build-Tools/

net.joala:joala-build-tools:jar:1.2.0

Support for Build-Tooling. Module can be taken as dependency to provide configuration data to reporting plugins for example like the checkstyle and PMD plugin.

Dependency

<dependency>
  <groupId>net.joala</groupId>
  <artifactId>joala-build-tools</artifactId>
  <type>jar</type>
  <version>1.2.0</version>
</dependency>

Usage

<plugin>
  <groupId>org.apache.maven.plugins</groupId>
  <artifactId>maven-checkstyle-plugin</artifactId>
  <dependencies>
    <dependency>
      <groupId>net.joala</groupId>
      <artifactId>joala-build-tools</artifactId>
      <version>${project.version}</version>
    </dependency>
  </dependencies>
</plugin>
